미러리스트를 연결상태에 따라 정렬 및 업데이트

Posted February 27, 2021 by Hackartist ‐ 1 min read

This page explains how to update and rank a mirrorlist in Arch linux.

목적

  • Arch 리눅스에서 미러리스트 업데이트
    • 업데이트시에 최신 미러리스트를 다운로드
    • 다운로드된 미러리스트를 연결 상태에 따라 정렬

사전 요구필요사항

  • pacman-contrib 설치
    • rankmirros 명령어 포함
pacman -S pacman-contrib

미러리스트 다운로드 및 정렬

  • Pacman mirrorlist generator 사용
    • 체크하고 싶은것들 체크후에 Generate List 를 클릭
    • 해당 링크를 사용하여 아래와 같이 수행
      • 예제에서 사용된 링크는 한국과 일본의 미러리스트를 대상으로 하는 링크
Server = http://mirror.premi.st/archlinux/$repo/os/$arch
Server = http://mirror.anigil.com/archlinux/$repo/os/$arch
Server = http://mirrors.cat.net/archlinux/$repo/os/$arch
Server = http://ftp.tsukuba.wide.ad.jp/Linux/archlinux/$repo/os/$arch
Server = http://ftp.jaist.ac.jp/pub/Linux/ArchLinux/$repo/os/$arch
